---

## 🎯 Sobre o curso

Este curso tem como objetivo aprofundar o uso do Git no dia a dia de desenvolvimento, mostrando como gerenciar versões de forma mais eficiente.

Ao longo das aulas, são trabalhados conceitos como branches, merge, rebase e organização de commits, ajudando a manter o histórico do projeto mais limpo e controlado.
